About CodeRoad

CodeRoad is a coding education platform aimed at creating interactive coding lessons. CodeRoad makes it easy to create, share, edit, and publish video-game-like tutorials that run alongside your code in the editor. It currently works with Javascript & Python, with more languages to come.

Inside Your Code Editor

CodeRoad is a free plugin for the popular Atom Editor.
With the "atom-coderoad" plugin, code in a real editor environment alongside your other favorite plugins.

  • Atom-CodeRoad

Create your Own Tutorials

Build tutorials using tools developers are already familiar with: Markdown & Unit Tests. Make tutorials is made easy with the new "builder-coderoad" plugin.

  • Builder-CodeRoad

Publish Free & Open

Version control & share your tutorials over NPM. Iterate and publish updates. Use Git to build & collaborate on courses.
